# 项目模块说明

该目录包含项目的所有核心源代码。

## 目录导航

- [**`config/`**](./config): 项目全局配置。
- [**`data/`**](./data): 数据持久化层，包含数据库连接和 SQL 脚本。
- [**`middlewares/`**](./middlewares): Express 中间件。
- [**`routes/`**](./routes): API 路由定义。
- [**`services/`**](./services): 业务逻辑封装。
- [**`utils/`**](./utils): 通用工具函数。

## 文件说明

- **`app.js`**: Express 应用程序配置和初始化。
- **`server.js`**: 项目入口文件，负责启动 HTTP 服务器。
